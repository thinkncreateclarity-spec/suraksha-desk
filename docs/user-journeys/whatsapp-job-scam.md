# WhatsApp job scam journey

1. Intake suspicious WhatsApp job message.
2. Capture screenshots, number, links, and payment asks.
3. Classify as likely recruitment scam / fraudulent communication.
4. Show safety guidance.
5. Route to Chakshu / Sanchar Saathi, and NCRP if financial harm exists.
6. Generate complaint-ready draft.
7. Track next steps.
