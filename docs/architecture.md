# Architecture

## Layers
1. Experience layer - web application for guided intake and follow-through.
2. API layer - incident-centric backend endpoints.
3. Domain layer - incident, evidence, route, draft, task, and timeline entities.
4. Evidence intelligence - extraction and normalization of signals.
5. Routing engine - deterministic official-channel routing plus urgency rules.
6. AI orchestration - classification, summarization, draft generation, guidance.
7. Tracking layer - non-government follow-up state and reminders.

## Architecture principle
AI proposes and explains. Rules constrain, validate, and route.
