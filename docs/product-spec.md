# Product spec

## Vision
Suraksha Desk is an AI-guided cyber incident assistant for India.

## Positioning
A navigation, evidence-preparation, complaint-drafting, and follow-through SaaS layer above official reporting rails.

## Not this product
- Not a government portal.
- Not an impersonation of NCRP, 1930, Sanchar Saathi, or Chakshu.
- Not a direct auto-filing proxy in the MVP.

## Core problem families
1. Fraud via phone, UPI, WhatsApp, URL, or paylink.
2. Password theft / account takeover.
3. Email job scam / recruitment phishing.

## Core flow
Incident detected -> classify -> collect evidence -> route -> draft complaint -> user submits -> track next steps
